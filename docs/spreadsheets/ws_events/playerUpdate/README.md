# `playerUpdate` → annotation spreadsheet

This folder documents the `playerUpdate` websocket event as a sortable
spreadsheet (LibreOffice Calc or any CSV viewer).

## Files

- `data_dictionary.csv` — your canonical, manually-annotated dictionary
- `data_dictionary.generated.*.csv` — autogenerated path inventories from samples

## Raw examples

Raw event envelopes are stored under `examples/` for canonical reference.
