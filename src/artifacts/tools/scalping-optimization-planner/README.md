# Scalping Optimization Planner (Prototype Dev Lab)

Offline HTML artifact for planning optimization experiments before execution.

## Purpose

- Build explicit run matrices for entry-first search
- Size drift-based exit permutation sweeps
- Estimate Monte Carlo simulation scale
- Generate kickoff markdown from current controls
- Keep exploration sequencing visible and reproducible
